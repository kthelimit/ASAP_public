창고 자재 등록

각 자재의 기초 재고를 등록할 수 있습니다. 같은 자재의 경우 마지막에 등록한 수량으로 덮어씌워집니다.
가용재고 = 기초재고 + 입고 예정 재고 - 출고 예정 재고

일괄 등록

엑셀 파일로 일괄 등록하는 경우, 템플릿을 받아 내용을 기입한 뒤 해당 파일을 선택하고 데이터 업로드를 눌러주세요. 이미 등록된 데이터들을 다운로드 받는 것도 가능합니다.
