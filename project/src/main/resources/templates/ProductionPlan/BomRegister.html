상품 등록

직접 등록

직접 등록하는 경우 아래의 폼을 기입 후 등록 버튼을 누르세요.

일괄 등록

엑셀 파일로 일괄 등록하는 경우, 템플릿을 받아 내용을 기입한 뒤 해당 파일을 선택하고 데이터 업로드를 눌러주세요. 이미 등록된 데이터들을 다운로드 받는 것도 가능합니다.

조립 구조 등록

조립구조는 현재 엑셀 파일로 일괄 등록만 할 수 있습니다. 템플릿을 받아 내용을 기입한 뒤 해당 파일을 선택하고 데이터 업로드를 눌러주세요.

BOM 등록

상품을 선택 시 맨 아래의 BOM 리스트에 해당 상품에 등록되어 있는 BOM의 내용이 뜹니다.

직접 등록

선택되어 있는 상품에 추가할 부품 종류를 선택 후, 자재를 골라 수량을 입력하고 등록하면 BOM을 추가할 수 있습니다.

일괄 등록

엑셀 파일로 일괄 등록하는 경우, 템플릿을 받아 내용을 기입한 뒤 해당 파일을 선택하고 데이터 업로드를 눌러주세요. 이미 등록된 데이터들을 다운로드 받는 것도 가능합니다.
